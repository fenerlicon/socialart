# Socialart Ajans Web App & CRM

Bu proje, Socialart Ajans için tasarlanmış modern bir landing page (funnel) ve MİY (Müşteri İlişkileri Yönetimi / CRM) / İş Takip paneli içermektedir.

## Özellikler
- **Landing Page:** Premium karanlık tema, entegre takvimli dinamik randevu formu ve hizmet filtresi.
- **Admin MİY Paneli:** Potansiyel Müşteri Hunisi (Lead Pipeline), Aktif Proje Takibi ve Personel Görev Panosu.
- **Güvenlik:** Role tabanlı yerel izolasyona sahip Login modülü.

## Teknik Altyapı
- React (Vite)
- Lucide React (İkonlar)
- React Router DOM
